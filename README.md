# dashGPT

Interface with AI gpt that prioritizes data sent in pdf format.

## Installation

```
git clone git@github.com:rosaldo/dashGPT.git
```

```
virtualenv dashGPT/venv
```

```
source dashGPT/venv/bin/active
```

```
pip install -r dashGPT/requirements.txt
```

## Run this app

```
./dashGPT/dash_gpt.py
```
